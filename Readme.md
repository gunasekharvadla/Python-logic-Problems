# Python Logic Problems
 
A structured Python practice repository progressing from fundamentals to advanced problem-solving, built for developers strengthening their logical thinking and interview readiness.
 
---
 
## Overview
 
This repository documents a hands-on Python learning journey through categorized problem sets. Each problem is isolated in its own file, written with clarity and real-world applicability in mind — not just syntactic correctness.
 
**Core objectives:**
- Develop systematic problem-solving habits
- Reinforce Python fundamentals through daily practice
- Build interview-ready coding confidence
- Establish consistent, professional coding discipline
---
 
## Repository Structure
 
```
PythonLogicProblems/
│
├── Conditions/          # if, elif, else, nested conditions
├── ExceptionHandling/   # try/except, custom exceptions
├── FileHandling/        # read, write, append, file operations
├── Functions/           # definition, scope, recursion
├── List/                # operations, comprehensions, sorting
├── Loops/               # for, while, nested loops, control flow
├── Strings/             # manipulation, formatting, validation
├── TupleDictionary/     # immutability, key-value operations
└── UserInput/           # input handling, type conversion
```

## Topics Covered
 
| Category | Concepts |
|---|---|
| **Variables & Datatypes** | Declaration, type conversion, type checking |
| **User Input** | `input()`, casting, validation |

| **Conditions** | `if/elif/else`, nested conditions, ternary |

| **Functions** | Parameters, return values, recursion, scope |

| **Lists** | CRUD operations, sorting, comprehensions |

## Practice Philosophy
 
- One problem per file — clean and focused
- Functions used for every logical unit
- Solutions written for readability before optimization
- Each problem tested with multiple edge-case inputs
- Code commented where logic is non-obvious
